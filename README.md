# restful-api

一个 node.js 的 RESTFUL API 服务，项目数据存储基于数据库 MongoDB，可以作为一个后端服务的基础结构持续迭代。

```
npm install
```

启动命令：

```
npm run start
```

调试命令：

```
npm run debug
```

API 文档地址：`http://127.0.0.1:8100/`
